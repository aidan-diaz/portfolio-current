# Sound effects

Optional 8-bit `.wav` files. If a file is missing, `useSound` automatically
falls back to a synthesized chiptune beep, so the site is fully functional
without any of these.

Drop in CC0 / royalty-free assets named exactly:

| Name          | When it plays                          |
| ------------- | -------------------------------------- |
| `hover.wav`   | Pointer hovers a `PixelButton`.        |
| `click.wav`   | Any button activation.                 |
| `start.wav`   | Intro screen "PRESS START" dismiss.    |
| `success.wav` | Contact form submitted successfully.   |
| `error.wav`   | Contact form submission error.         |

Good free sources:

- https://opengameart.org/ (filter by "Sound Effect" + CC0)
- https://freesound.org/ (filter by CC0)
- https://www.bfxr.net/ (generate chiptune SFX in-browser, export to .wav)

Keep clips short (40–250ms) and quiet (peaks around -12 dBFS).
